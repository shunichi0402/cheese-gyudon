
# cheese-gyud
## 電子出席簿
## サーバーの建て方

node.jsをインストールする
一緒にnpmもインストールされる。
https://nodejs.org/ja/

gitをインストールする。
https://gitforwindows.org/

コマンドプロンプトを開き作業ディレクトリに移動して、以下のコードを打つ
```
$ git clone https://github.com/shunichi0402/cheese-gyudon.git
$ cd cheese-gyudon
$ npm i
```

つぎに以下のコードでサーバーが立ち上がる。
```
npm run dev
```
ブラウザで http://localhost:3000 にアクセスすれば表示される。
